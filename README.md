# catalagoslv# Catálogo de Produtos - Leonardo A Verza

Um catálogo web profissional e responsivo para gerenciar produtos de informática.

## 🌐 Acesse

[https://seu-usuario.github.io/catalogo-produtos-informatica/](https://seu-usuario.github.io/catalogo-produtos-informatica/)

## ✨ Funcionalidades

- ✅ Adicionar novas peças
- ✅ Remover peças com confirmação
- ✅ Tabela responsiva
- ✅ Links externos para conferência
- ✅ Categorias pré-definidas
- ✅ Toast notifications para feedback
- ✅ Design moderno e clean
- ✅ Total de peças em tempo real

## 🚀 Como usar

### Adicionar uma peça
1. Clique no botão **"Adicionar Peça"**
2. Preencha os dados (apenas *Produto/Serviço* é obrigatório)
3. Clique em **"Adicionar"**

### Remover uma peça
1. Clique no ícone de lixeira 🗑️ na linha da peça
2. Confirme a remoção no diálogo
